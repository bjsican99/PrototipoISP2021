Verificacin De Datos
Al ingresar en la ventana de Verificacin de Datos, primeramente, tendremos buscar el CUI.
Al momento de ingresarlo, presionamos el botn Verificar entonces nos aparecer los datos
correspondientes a dicha persona. Ver Imagen 1.
Imagen 1, Datos del CUI.
Al momento de darle en el botn de Verificar, el usuario tendr que validar los datos
manualmente.
Tendr que marcar la Casilla de Datos DPI Validos.
As tambin en la parte inferior a la casilla, aparece si la persona, ya cuenta con un pasaporte.
El usuario tendr que verificar el boleto de ornato, para ello lo tiene que ingresar, despus en
verificar.
Nota: para que se habilite el botn tiene que llenar la casilla Datos DPI Validos, para
que se pueda desbloquear.
Al momento de ya verificar el boleto, en el programa se podr observar el estado del boleto,
si esta valido se pondr verde y si es invalido se quedar en rojo.
Una vez validados los datos del DPI y el boleto de ornato, se habilitar el botn aceptar.

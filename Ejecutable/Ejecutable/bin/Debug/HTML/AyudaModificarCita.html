Innovación y Desarrollo Para El Futuro
1. Navegador
Este objeto nos ayuda a poder habilitar los campos de insertar, modificar,
eliminar, guardar, cancelar y poder ver las ayudas. El navegador nos
ayuda a poder movernos entre los campos ya existentes en el
DataGridView, y poder imprimir información de esta si poseen los
permisos.
2. Campos de datos
El campo de datos deberá ingresar el numero de cita para verificar si
existe, si es válida mostrará los datos ingresados durante la creación de
la cita y permitirá modificar la hora y fecha de la cita, los otros campos
no será posible editarlos.
3. DataGridView
Nos muestra los módulos ya existentes para poder seleccionarlos y
realizar las acciones que se deseen tanto como modificar y eliminar.
